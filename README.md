# Fototime AI Telegram Mini App

## О проекте

Fototime AI — это pet-проект Telegram Mini App для генерации AI-фотографий на основе пользовательского изображения, выбранной роли и визуального стиля.

Проект создан как демонстрация навыков Manual QA / Middle+ QA:

- анализ требований;
- проектирование пользовательских сценариев;
- описание негативных кейсов;
- тестовая стратегия;
- API-проверки;
- работа с Telegram Mini Apps;
- подготовка тестовой документации;
- проверка обработки ошибок на клиенте и сервере.

## Основной пользовательский сценарий

1. Пользователь открывает Mini App из Telegram-бота.
2. Выбирает роль.
3. Выбирает стиль изображения.
4. Загружает фото.
5. Запускает генерацию.
6. Получает результат.
7. Может создать новое изображение.

## Запуск проекта локально

1. Установить зависимости:

npm install

2. Создать .env:

cp .env.example .env

3. Запустить проект:

npm run dev

4. Открыть приложение:

http://localhost:3000
