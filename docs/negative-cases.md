# Negative Cases

## Event Configuration

| ID | Case | Expected Result |
|---|---|---|
| NEG-001 | Конфигурация мероприятия не загрузилась | Отображается error-state |
| NEG-002 | Backend вернул 404 по `/api/event-config` | Пользователь видит сообщение о недоступности мероприятия |
| NEG-003 | Мероприятие неактивно | Генерация недоступна |
| NEG-004 | Список участников пустой | Отображается empty-state |
| NEG-005 | Список стилей пустой | Отображается empty-state |
| NEG-006 | У выбранного участника нет доступных стилей | Отображается сообщение об отсутствии стилей |

## Participant and Style Selection

| ID | Case | Expected Result |
|---|---|---|
| NEG-007 | Не выбран участник | Генерация недоступна |
| NEG-008 | Не выбран стиль | Генерация недоступна |
| NEG-009 | Пользователь меняет участника после выбора стиля | Неподходящий стиль сбрасывается |
| NEG-010 | Стиль не соответствует выбранному участнику | Backend возвращает `STYLE_PARTICIPANT_MISMATCH` |
| NEG-011 | Стиль недоступен | Backend возвращает `INVALID_STYLE` |
| NEG-012 | Участник недоступен | Backend возвращает `INVALID_PARTICIPANT` |
