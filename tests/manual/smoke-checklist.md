# Smoke Checklist

## Application Start

- [ ] Open the application.
- [ ] Check that the main screen is displayed.
- [ ] Check that event status is displayed.
- [ ] Check that event configuration is loaded.

## Main Flow

- [ ] Select participant.
- [ ] Check that styles are filtered by selected participant.
- [ ] Select style.
- [ ] Upload a valid JPG or PNG image.
- [ ] Check that the generation button becomes active.
- [ ] Click the generation button.
- [ ] Check that loading state is displayed.
- [ ] Check that success message is displayed.
- [ ] Check that result section is displayed.
- [ ] Reset the flow.
- [ ] Check that form state is cleared.
